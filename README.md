#  TechFix - Venta y Servicio Técnico

Sitio web desarrollado con **HTML5** y **CSS3 puro** para un negocio de venta, reparación y mantenimiento de computadoras y notebooks.

#  Descripción

TechFix es una tienda de tecnología en Buenos Aires que ofrece:
- Venta de computadoras de escritorio y notebooks
- Reparación y service técnico
- Mantenimiento preventivo
- Upgrades y mejoras de hardware
- Venta de componentes y periféricos

## Cómo usar

1. Clonar o descargar el repositorio
2. Abrir `index.html` en el navegador
3. Para publicar: subir a GitHub Pages
